# Teda Foods

Fresh jollof rice, fried rice, pasta & more — made daily at FUNAAB, Abeokuta. Order online or via WhatsApp.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Lint & Format

```bash
npm run lint
npm run format
```

## Environment Variables

| Variable                   | Description              |
| -------------------------- | ------------------------ |
| `VITE_PAYSTACK_PUBLIC_KEY` | Paystack live public key |

Copy `.env` and fill in your values.
